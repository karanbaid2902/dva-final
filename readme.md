# Master's E-Commerce Sales Dashboard

An advanced analytics dashboard built with **Streamlit** and **Plotly**. This project features a synthetic dataset engine that mimics real-world retail trends including seasonality and customer demographics.

## Features
- **5 Multi-dimensional Sections**: Executive, Sales, Customer, Product, and Regional.
- **18+ Interactive Charts**: Including funnels, sunbursts, and time-series growth.
- **RFM-ready Data**: Synthetic data includes unique Customer IDs and purchase frequency.

## How to Run
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`
3. Generate the data: `python data_generator.py`
4. Launch the dashboard: `streamlit run app.py`
